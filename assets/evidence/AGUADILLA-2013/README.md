# Aguadilla video copies

- `DOD_110692805-Aguadilla.mp4` is the 154,997,462-byte local official-release copy retained in the canonical research workspace.
- `DOD_110692805-Aguadilla-web.mp4` is the 49,878,228-byte H.264 CRF 28 derivative deployed for browser access.

The derivative preserves the 1920×1080 frame dimensions and approximately 234.5-second duration but is not byte-identical to the source copy. SHA-256 values are recorded in `CHECKSUMS.sha256`.
